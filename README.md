# calculator

>一个vue实现的简单计算器

# 技术栈
* [x] vue
* [x] webpack
* [x] AlloyFinger
* [x] vue computed

# 实现的效果
* [x] 左右滑动切换主题
* [x] 整数位大于3位自动添加逗号
* [x] 输入加减乘除符号直接计算结果
* [x] 清除所有数据
* [x] 退格
* [ ] 按钮动画
* [x] 在输出结果之后不能退格

# bug
* [x]  在手机段不是全屏固定的，可以拖动



## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --repor
```
